# Notice

This project created by `Vite`, used [`Vite 4`](https://vitejs.dev/), [`React 18.2`](https://reactjs.org/), [`antd 5.5`](https://ant.design/)

## if you used NPM
### install
```bash
npm i
```

### start 
```bash
npm run dev
```

### build / deploy
```bash
npm run build
```

## if you used Yarn
### install
```bash
yarn
```

### start 
```bash
yarn dev
```

### build / deploy
```bash
yarn build
```
